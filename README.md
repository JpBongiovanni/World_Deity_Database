### World Deity Database - Python/Django

A wiki-fandom inspired information aggregation tool for scholars and non-scholars alike which serves as a hub for specific information and sources in a niche field to aid researchers as they study ancient cultures. This application was developed full stack and deployed to Heroku. This web application has an intuitive backend and a responsive frontend built with Bootstrap, Python and Django framework. This site has functionality with PostgreSQL that enables advanced search engine queries and results while ensuring security with bcrypt encryption.
